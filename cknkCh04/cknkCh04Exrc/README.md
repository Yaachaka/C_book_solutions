# cknkCh04Exrc001

Section 4.1

<!-- START: Problem Statement -->

Show the output produced by each of the following program fragments. Assume that `i`, `j`, and `k` are `int` variables.

<ol type="a">
<li>

```C
i = 5; j = 3;
printf("%d %d", i / j, i % j);
```

</li>
<li>

```C
i = 2; j = 3;
printf("%d", (i + 10) % j);
```

</li>
<li>

```C
i = 7; j = 8; k = 9;
printf("%d", (i + 10) % k / j);
```

</li>
<li>

```C
i = 1; j = 2; k = 3;
printf("%d", (i + 5) % (j + 2) / k);
```

</li>
</ol>

<!-- END: Problem Statement -->
